This is an example of how to embed the JSOnline.com Projects footer. Place the following code snippet where you'd like the footer to be within the code. This assumes that the Futura fonts are already loaded on the page and that you have a mobile friendly viewport set. See this page's source as an example.

For projects.jsonline.com webpages:

<script id="footer-script" src="footer.js" 
	data-first-link=""
	data-first-headline=""
	data-first-image=""
	data-first-desc=""
	data-second-link=""
	data-second-headline=""
	data-second-image=""
	data-second-desc="">
</script>
